# MCPHub Desktop 1.0.16

## 新功能
- 支持 Windows ARM64 构建
- 支持 Linux ARM64 (.deb) 构建
- 支持 macOS ARM64 和 x64 构建

## 修复
- 修复 PowerShell 脚本在 CI 环境中的 StandardOutputEncoding 错误
- 修复 Rust Option<String> to_lowercase 编译错误
- 修复 Linux AppImage 打包在 CI 中不可靠的问题
- 修复 Windows ARM64 交叉编译时 Node.js 版本验证失败
- 升级 GitHub Actions 到最新版本，消除 Node.js 20 弃用警告

## 其他
- 产物文件名统一为 mcphub-desktop-{os}-{arch}.{ext} 格式
- Linux 仅提供 .deb 和 .rpm 包（移除 AppImage）
